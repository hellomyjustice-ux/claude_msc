## 컬러 사용 가이드

### Primary (`#5589A5`) — 스틸 블루
- 페이지 배경, 헤더, 섹션 배경에 사용
- 차분하고 신뢰감 있는 영역에 적용
- 어두운 변형(`#4A7A96`)은 호버 상태, 그림자, 오버레이에 활용

### Secondary (`#F0E3C8`) — 크림 아이보리
- 메인 헤딩, 대형 타이포그래피에 사용
- 블루 배경 위에서 높은 가독성 확보
- 카드, 패널의 배경색으로도 활용 가능

### Accent (`#D4961A`) — 골든 옐로우
- CTA 버튼, 링크 호버, 강조 텍스트에만 사용
- 전체 비율 3% 이내로 절제하여 포인트 효과 극대화
- 호버 시 어두운 변형(`#B87E14`) 적용

### Text Colors
- **헤딩**: `--color-text-primary` (`#F0E3C8`)
- **본문**: `--color-text-secondary` (`#F5F5F5`)
- **캡션/보조**: `--color-text-muted` (`#C8D8E0`)

## 반응형 고려사항

### 모바일 (xs ~ sm, 0px ~ 600px)
- 배경 블루(`#5589A5`)를 전체 화면에 유지
- 텍스트 대비비 확보: 크림(`#F0E3C8`) on 블루(`#5589A5`) → 대비비 약 4.5:1 (WCAG AA 충족)
- 골든 옐로우 버튼은 최소 44px 터치 영역 확보

### 태블릿 (md, 600px ~ 900px)
- 카드/패널 배경에 `#F5EEE0` 적용하여 섹션 구분
- 타이포그래피 크기 확대 시 크림 컬러 유지

### 데스크톱 (lg 이상, 900px+)
- 밝은 영역과 블루 영역을 구분하는 레이아웃 분할 가능
- 사이드바나 네비게이션에 `#4A7A96` 어두운 변형 적용
- 골든 옐로우 액센트를 섹션 구분선, 아이콘 포인트에 활용

## 디자인 특징

**전체 분위기**: 차분하고 세련된 **내추럴 미니멀리즘** 톤

**브랜드 느낌**: 고급 라이프스타일 · 뷰티 · 웰니스 브랜드에 최적화  
차가운 블루와 따뜻한 크림의 대비가 현대적이면서도 자연친화적인 이미지를 만들어냅니다.

**핵심 원칙**: 골든 옐로우(`#D4961A`)는 전체의 3% 미만으로 유지할 때 시선 집중 효과가 극대화됩니다.
