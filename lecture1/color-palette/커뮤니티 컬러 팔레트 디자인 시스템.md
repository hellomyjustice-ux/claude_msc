## 컬러 사용 가이드

### Primary (`#2B3A4E`) — 다크 네이비
- 메인 헤딩, 네비게이션, 주요 텍스트에 사용
- 베이지 배경 위에서 높은 대비로 가독성 확보 (대비비 약 8:1, WCAG AAA 충족)
- 라이트 변형(`#3D5060`)은 서브 텍스트, 아이콘에 활용

### Secondary (`#E8D8B0`) — 샌디 베이지
- 페이지 배경, 카드 배경, 화이트 대체 색상으로 사용
- 순백 대신 웜 톤 베이지를 사용해 따뜻하고 빈티지한 분위기 연출
- 어두운 변형(`#D4C49A`)은 섹션 구분, 보더, 그림자에 활용

### Accent (`#B07A2A`) — 앰버 브론즈
- CTA 버튼, 링크 호버, 강조 텍스트에만 사용
- 전체 비율 3% 이내로 절제하여 포인트 효과 극대화
- 호버 시 어두운 변형(`#8E6220`) 적용

### Panel (`#456070`) — 슬레이트 블루
- 사이드바, 다크 카드, 오버레이 배경에 사용
- 내부 텍스트는 반드시 오프 화이트(`#F0ECE0`) 사용
- 메인 베이지 영역과의 대비로 레이아웃 분할에 효과적

### Text Colors
- **메인 헤딩**: `--color-text-primary` (`#2B3A4E`)
- **본문/서브**: `--color-text-secondary` (`#3D5060`)
- **캡션/보조**: `--color-text-muted` (`#7A8E9C`)
- **다크 배경 위**: `--color-text-on-dark` (`#F0ECE0`)

## 반응형 고려사항

### 모바일 (xs ~ sm, 0px ~ 600px)
- 배경 베이지(`#E8D8B0`)를 전체 화면에 유지
- 텍스트 대비비 확보: 네이비(`#2B3A4E`) on 베이지(`#E8D8B0`) → 대비비 약 8:1 (WCAG AAA 충족)
- 앰버 브론즈 버튼은 최소 44px 터치 영역 확보
- 슬레이트 패널(`#456070`)은 모바일에서 전체 너비 배너로 활용

### 태블릿 (md, 600px ~ 900px)
- 좌측 베이지 영역 + 우측 슬레이트 패널 2단 레이아웃 적용 가능
- 카드/패널 배경에 어두운 베이지(`#D4C49A`)로 섹션 구분

### 데스크톱 (lg 이상, 900px+)
- 포스터 원본처럼 베이지 배경 + 우측 슬레이트 사이드바 레이아웃 구성
- 앰버 브론즈 액센트를 섹션 구분선, 아이콘 포인트, 호버 효과에 활용
- 씬 블루 그레이(`#7A8E9C`)를 이미지 오버레이, 배경 그라데이션에 사용

## 디자인 특징

**전체 분위기**: 빈티지하면서도 현대적인 **저패니즈 트래블 에디토리얼** 톤

**브랜드 느낌**: 여행 · 문화 · 라이프스타일 · 커뮤니티 플랫폼에 최적화  
웜 베이지와 딥 네이비의 대비가 클래식하고 신뢰감 있는 이미지를 형성하며, 앰버 브론즈 포인트가 동양적인 고급미를 더합니다.

**핵심 원칙**: 베이지 배경은 흰색보다 눈의 피로도가 낮아 장문의 컨텐츠가 많은 커뮤니티 UI에 적합합니다. 앰버 브론즈(`#B07A2A`)는 3% 이내로 절제할 때 강조 효과가 극대화됩니다.
