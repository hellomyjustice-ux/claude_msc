## 1. 타이포그래피 (Typography)

### 폰트 계층 구조

| 레벨 | 용도 | font-size | font-weight | line-height |
|------|------|-----------|-------------|-------------|
| Display | 히어로 대형 타이틀 | `96px` (6rem) | 700 | 1.0 |
| H1 | 페이지 대표 제목 | `36px` (2.25rem) | 700 | 1.2 |
| H2 | 섹션 제목 | `24px` (1.5rem) | 600 | 1.3 |
| H3 | 카드/패널 제목 | `18px` (1.125rem) | 600 | 1.4 |
| H4 | 서브 타이틀 | `16px` (1rem) | 600 | 1.4 |
| Body 1 | 기본 본문 | `14px` (0.875rem) | 400 | 1.6 |
| Body 2 | 보조 본문 | `13px` (0.8125rem) | 400 | 1.6 |
| Caption | 레이블 / 날짜 | `12px` (0.75rem) | 400 | 1.5 |
| Overline | 태그 / 배지 | `11px` (0.6875rem) | 500 | 1.4 |

### 분석 근거

**BLOSSOM 포스터:**
- 포스터 전체를 채우는 디스플레이 타입 → `Display` 레벨 정의
- 하단 키워드 "NEW BEGINNINGS · SIMPLICITY · PURITY" → 자간 넓은 `H4` 패턴
- 우측 단락 텍스트 → `Caption` 수준 소형 본문 패턴

**Crextio 대시보드:**
- "Welcome in, Nixtio" 페이지 제목 → `H1` 레벨 (36px / 700)
- "Progress", "Time tracker" 섹션 제목 → `H3` 레벨 (18px / 600)
- 통계 숫자 "92", "315" → `H1` 사이즈 + 700 weight
- 네비게이션, 리스트 항목 → `Body 2` 레벨 (13~14px)
- 날짜, 시간 레이블 → `Caption` 레벨 (12px)

## 2. 여백 시스템 (Spacing)

### 기본 간격 단위

4px 기본 단위 기반 스케일 사용 (MUI 기본값과 동일)

| 토큰 | 값 | px | 용도 |
|------|----|----|------|
| `spacing(0.5)` | 0.5 | `2px` | 아이콘 내부 미세 간격 |
| `spacing(1)` | 1 | `4px` | 텍스트 간 최소 간격 |
| `spacing(2)` | 2 | `8px` | 아이콘-텍스트 간격, 배지 내부 패딩 |
| `spacing(3)` | 3 | `12px` | 리스트 아이템 간격, 인라인 간격 |
| `spacing(4)` | 4 | `16px` | 카드 내부 요소 간격, 섹션 내 gap |
| `spacing(5)` | 5 | `20px` | 카드 패딩 (내부 여백) |
| `spacing(6)` | 6 | `24px` | 카드 간 gap, 섹션 패딩 |
| `spacing(8)` | 8 | `32px` | 섹션 간 간격 |
| `spacing(10)` | 10 | `40px` | 페이지 상하 패딩 (모바일) |
| `spacing(12)` | 12 | `48px` | 페이지 상하 패딩 (데스크톱) |

### 분석 근거

**Crextio 대시보드:**
- 카드 내부 패딩: `20~24px` → `spacing(5~6)`
- 카드 사이 gap: `16px` → `spacing(4)`
- 네비게이션 높이: `56px`, 상하 패딩 `12px` → `spacing(3)`
- 리스트 항목 간격: `12px` → `spacing(3)`
- 통계 레이블-값 간격: `4~8px` → `spacing(1~2)`
- 페이지 외부 여백: `24px` → `spacing(6)`

**BLOSSOM 포스터:**
- 전체 여백 없는 풀블리드 → 포스터/히어로 영역은 `py: 0`
- 하단 키워드 영역 상단 여백: `spacing(6~8)`
- 우측 텍스트 블록 줄 간격: `spacing(3~4)`

### CSS 변수 정의 — 여백

```css
:root {
  /* Spacing */
  --spacing-1: 4px;
  --spacing-2: 8px;
  --spacing-3: 12px;
  --spacing-4: 16px;
  --spacing-5: 20px;
  --spacing-6: 24px;
  --spacing-8: 32px;
  --spacing-10: 40px;
  --spacing-12: 48px;

  /* Component-specific Spacing */
  --padding-card: var(--spacing-5);          /* 20px */
  --gap-card: var(--spacing-4);              /* 16px */
  --padding-section: var(--spacing-6);       /* 24px */
  --gap-section: var(--spacing-8);           /* 32px */
  --padding-page-x: var(--spacing-6);        /* 24px */
  --padding-page-y: var(--spacing-10);       /* 40px */
  --padding-button: var(--spacing-2) var(--spacing-4);  /* 8px 16px */
  --padding-chip: 6px var(--spacing-3);      /* 6px 12px */
  --height-nav: 56px;
}
```

### 반응형 여백 규칙

```jsx
/* 페이지 여백 */
py: { xs: 2, md: 4, lg: 6 }   /* 8 / 16 / 24px */
px: { xs: 2, md: 3, lg: 6 }   /* 8 / 12 / 24px */

/* 카드 내부 패딩 */
p: { xs: 2, md: 3 }            /* 8 / 12px */

/* 섹션 간 간격 */
gap: { xs: 2, md: 3, lg: 4 }  /* 8 / 12 / 16px */
```

## 3. 레이아웃 (Layout)

### 그리드 시스템

| 구분 | 컬럼 수 | Gutter | 최대 너비 |
|------|---------|--------|-----------|
| 모바일 (xs, 0px~) | 4열 | 8px | 100% |
| 태블릿 (sm, 600px~) | 8열 | 16px | 100% |
| 데스크톱 (md, 900px~) | 12열 | 24px | 1280px |
| 와이드 (lg, 1200px~) | 12열 | 24px | 1440px |

### 대시보드 레이아웃 구조

```
┌─────────────────────────────────────────┐
│          Navigation (height: 56px)       │  ← 전체 폭, z-index: 10
├─────────────────────────────────────────┤
│  페이지 제목 + 요약 통계 바         (H1)  │  ← col 12
├──────────────┬─────────────┬────────────┤
│ 프로필 카드  │  Progress   │ Onboarding │
│  (col 3)    │  (col 5)    │  (col 4)   │
├─────────────┼─────────────┤            │
│ 사이드 목록 │ Time tracker│            │
│  (col 3)    │  (col 5)   │            │
├─────────────┴─────────────┴────────────┤
│         캘린더 / 일정 (col 12)          │
└─────────────────────────────────────────┘
```

### 포스터 레이아웃 구조

```
┌──────────────────────────────┐
│  (우상단) 소형 텍스트 블록    │  ← position: absolute, top/right
│                              │
│   디스플레이 타입 (BLOSSOM)  │  ← full bleed, overflow hidden
│                              │
│   [중앙 이미지 / 오브젝트]   │  ← absolute center
│                              │
│   하단 키워드 텍스트 줄       │  ← bottom, letter-spacing: wide
└──────────────────────────────┘
```

### Border Radius

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--radius-sm` | `6px` | 배지, 칩, 소형 버튼 |
| `--radius-md` | `10px` | 버튼, 인풋 필드 |
| `--radius-lg` | `14px` | 카드, 패널 |
| `--radius-xl` | `20px` | 다크 카드, 프로필 카드 |
| `--radius-full` | `9999px` | 네비게이션 활성 탭, 아바타 |
